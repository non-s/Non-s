## Stack

Todos os projetos: HTML, CSS e JavaScript puros. Sem frameworks, sem etapas de build, sem bundlers. Cada repositório tem exatamente 4 arquivos: `index.html`, `style.css`, `script.js`, `README.md`.

A restrição é intencional. Saber o que a plataforma oferece nativamente — WebRTC, Canvas, IntersectionObserver, localStorage, Web Share API — importa mais do que saber qual camada de abstração instalar.
