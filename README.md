## Stack

Every project: plain HTML, CSS, and JavaScript. No frameworks, no build steps, no bundlers. Each repo is exactly 4 files: `index.html`, `style.css`, `script.js`, `README.md`.

The constraint is intentional. Knowing what the platform gives you for free — WebRTC, Canvas, IntersectionObserver, localStorage, the Web Share API — matters more than knowing which abstraction layer to install.
